Команды линух:
cd -изменяет текущую директорию
mkdir -создает папку 
touch - создает файлы
ls -показывает содержимое текущей дтректории
rm - удаляет каталоги и файлы
mv - перемещает файлы в другие директории
cp - копирует файлы
cd ../ -возвращает на одну папку назад 
clear - просто очищает
history - просто показывает команды
pwd - выводит полный путь текущей директории
find - ищет файлы по определенным критериям
rename - переименовывает файлы
sudo - дает права суперпользователя
locate - находит нужный файл

Git команды:
git init - создание локального репозитория
git add . - добавляет содержимое рабочей директории в индекс
git commit - фиксирует изменения
git remote add - добавление удаленного репозитория
git push - заливает файл в удаленный репозиторий
git branch - показывает существующие ветки
git clone - клонирует удаленного репозитория
git status - показывает состояние файлов в рабочей директории и индексе
git merge - используется для слияние веток
