Linux commands:
cd-поменять директорию
ls-посмотреть все файлы
pwd-выводит текущую рабочую директорию
mkdir-создает новую папку
rm-удаляет файлы или директории
mv-переименовывает
cp-copy
cat-показывает содержимое файла
clear-очищает
history-посмотреть историю
cd ..-вернуться на уровень выше
cd ../../-вернуться на два уровня выше
touch-создает новый файл

Git commands:
git status-показывает состояние файлов в рабочем каталогеи индексе
git add . - добавляет изменение из рабочего каталога в раздел проиндексированныхфайлов
git branch-показывает список всех веток в репозитории
git init-инициализирует новый репозиторий гит в текущем каталоге
git commit-фиксирует изменения сделанные в рабочем каталоге и добавляет их в репозиторий
git remote add origin-используется для связывания вашего локального репозитория Гит с удалённым репозиторием
git push origin master-используется для отправки изменений из локального репозитория в удалённый репозиторий

New commands:
wc-выводит число строк, слов и символов в файле.Пример использования: wc -1 file.txt-подсчитает количество строк в файле file.txt

grep-команда для поиска текста в файлах или потоках данных. Например: "pattern" file.txt-найдет строки в файле file.txt, содержающие "pattern"

sort-сортирует строки текстовых файлов. Пример Использования: sort file.txt-отсортирует строки файла file.txt в алфавитном порядке

find-находит файлы и каталоги на основе различных критериев. Пример использования: find .-name "*.txt"-найдет все файлы с расширением .txt в текущем каталоге и его подкаталогах.

awk-мощный язык для обработки и анализа текстовых файлов. Может использовать дляманипуляции с данными, сравнения строк и многое другое. Пример использования:  awk '{print NF}' file.txt-выведет количество полей в каждой строке файла file.txt

Git commands (new)
git log-показывает историю коммитов. Пример использования: git log-выведет список всех коммитов в репозитории
git stash-временно сохраняет изменения, чтобы переключиться на другую ветку или выполнить другие операции. Примеры использования: git stash-сохранит текущие изменения в стэше
